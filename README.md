# CodeAlpha Stock Portfolio Tracker

## Description
This project is developed as part of the CodeAlpha Python Programming Internship. It calculates the total investment value of stocks entered by the user using predefined stock prices.

## Features
- User enters stock names and quantities
- Calculates total investment value
- Saves the result in a text file
- Simple and easy-to-use console application

## Technologies Used
- Python

## How to Run

1. Save the file as `stock_portfolio.py`.
2. Open the terminal in the project folder.
3. Run:

```bash
python stock_portfolio.py
```
